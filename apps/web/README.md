# Lotion - Frontend app

## Overview

Main frontend application for Lotion

## Proyect structure

```js
+-- public // public assets
|   +-- images // .png or .jpeg images
|   +-- icons // .svg icons
+-- src // main React.js source code
|   +-- components // global components
|   +-- pages // global pages
|   +-- styles // global styles
|   +-- router.tsx // application router
+--
```
